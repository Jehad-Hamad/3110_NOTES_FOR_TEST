# CS3110 Lab4 - WebGL Graphics Exercises

This lab contains three WebGL exercises demonstrating different graphics concepts and techniques.

## Exercises

### Exercise 1: Sierpinski triangle of self and different filtering options

- **File**: `Exercise1.html` + `Exercise1.js`
- **Description**: Sierpinski Triangle of me. Bottom left is CLAMPING, Upper middle is REPEAT, Bottom right is MIRROR REPEAT
- **Controls**:
  - Arrow Up: Increase recursion level (more detail)
  - Arrow Down: Decrease recursion level (less detail)
  - Z: Start/stop rotation animation

### Exercise 2: Sperinski Triangles and add a different texture (Multi Texturing)

- **File**: `Exercise2.html` + `Exercise2.js`
- **Description**: Sperinski Triangles and add a different texture (Multi Texturing). Bottom left is GRASS, Upper middle is WOOD, Bottom right is METAL
- **Controls**:
  - Arrow Up: Increase recursion level (more detail)
  - Arrow Down: Decrease recursion level (less detail)
  - Z: Start/stop rotation animation

## How to Run

1. Open any of the HTML files in a web browser that supports WebGL
2. Use the keyboard controls to interact with each exercise
3. Press Z to toggle animation on/off
